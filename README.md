# 🔐 Protocol → Tamarin Rule Generator

This project converts protocol steps into Tamarin Prover rules automatically.

## 🚀 Features
- Convert protocol text into formal rules
- Supports multi-step communication
- Download `.spthy` file
- Clean UI using Streamlit

## 🧠 Example Input
User -> Server: Login  
Server -> User: Token  

## 🛠 Tech Stack
- Python
- Streamlit

## ▶️ How to Run
```bash
pip install streamlit
streamlit run app.py
